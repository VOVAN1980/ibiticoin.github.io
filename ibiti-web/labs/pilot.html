The problem

AI is probabilistic. Money is not.

Agents hallucinate, get prompt-injected, and mis-execute trades. Giving any agent unlimited approvals or raw key access turns a single mistake into irreversible loss.

The fix

Deterministic constraints on-chain.

Spend limits per-tx and rolling windows — enforced in execution, not in UI.
Allowlist / denylist targets and call keys — block unknown interactions by default.
Instant revoke + TTL/nonce/deadline — kill replay and stale signatures.
