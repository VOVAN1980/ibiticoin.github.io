THE PROBLEM

AI is probabilistic. Money is not.

Agents hallucinate, get prompt-injected, or mis-execute trades. If an agent has a key (or unlimited approvals), one mistake becomes irreversible loss.

THE FIX

Deterministic constraints on-chain.

EPK enforces policies: spend caps, allowlists, deadlines, nonces, and validator rules. If an agent violates policy — transaction reverts. No “trust me bro”.
