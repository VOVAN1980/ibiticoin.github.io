The Execution Pipeline

Logic is separated from execution. The Kernel is immutable; logic is modular.

1

Sign Intent

Agent constructs intent. Owner key signs an EIP-712 message off-chain.

2

Validate

Kernel verifies signature & Nonce. Delegates to Validator modules for logic checks.

3

Execute

If passed, call is executed via call(). Reverts instantly on failure.
