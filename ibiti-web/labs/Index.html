The Execution Pipeline

Logic is separated from execution. The Kernel is immutable; logic is modular. Update your risk rules without migrating funds.

1

Sign Intent (Off-chain)

Your AI agent constructs a transaction intent. The Owner key signs an EIP-712 typed message. Private keys never touch the internet.

2

Validate (On-chain)

The Kernel verifies the signature and Nonce. It then delegates to Validators (SpendLimit, Allowlist, ThreatFeed) to enforce business logic constraints.

3

Execute (Finality)

If all checks pass, the call is executed via call(). Any violation causes an instant revert, protecting the treasury.
